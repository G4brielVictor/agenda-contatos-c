# Agenda de Contatos em C

## 📌 Descrição
Sistema de agenda de contatos desenvolvido em C, com armazenamento em arquivo.

## ⚙️ Funcionalidades
- Cadastrar contatos
- Listar contatos
- Buscar por nome e ID
- Editar contatos
- Remover contatos

## 💾 Armazenamento
Os dados são salvos em arquivo local (.dat).

## ▶️ Como executar

```bash
gcc src/*.c -Iinclude -o agenda
./agenda
